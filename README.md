## README для проекта "SiteConstructor"

# SiteConstructor

Программа "SiteConstructor" разработана на языке Go для создания и управления веб-сайтами с использованием функционала, аналогичного тому, который описан на сайте [бесплатный конструктор сайтов](https://prohoster.info/hosting/constructor).

### Описание

"SiteConstructor" позволяет пользователям:

- Создавать сайты с заданными настройками.
- Управлять страницами: добавлять, редактировать и удалять.
- Получать информацию о подключении к хостингу.

### Установка

1. Убедитесь, что у вас установлен [Go](https://golang.org/dl/).
2. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/username/repo.git
   cd repo
   ```

3. Запустите программу:

   ```bash
   go run main.go
   ```

### Использование

Программа будет доступна по адресу [http://localhost:8080](http://localhost:8080). При переходе по этому адресу вы увидите информацию о созданном сайте и его страницах.

### Примечание

Для дополнительной информации о хостинге и его возможностях, посетите [бесплатный конструктор сайтов](https://prohoster.info/hosting/constructor).
